# SOCIAL LINKS PROFILE
This project is a personal landing page that aggregates all my social media and professional profiles in one place. It's a convenient way to share my online presence with others, showcasing my professional and personal interests across various platforms.

## Features
- **Social Media Links:** Direct links to my profiles on platforms like LinkedIn, GitHub, Twitter, and more.
- **Responsive Design:** The page is fully responsive, providing an optimal viewing experience on both desktop and mobile devices.

## Technologies Used

- **HTML:** For structuring the content of the page.
- **CSS:** For styling and layout, ensuring a cohesive and modern design.
